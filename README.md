## 🛠️ Tech Stack

This portfolio is built with modern web technologies, focusing on performance, aesthetics, and smooth user experiences:

*   **Framework:** [Next.js](https://nextjs.org/) (React framework utilizing the App Router)
*   **Styling:** [Tailwind CSS v3](https://tailwindcss.com/) (Utility-first CSS framework for rapid UI development)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/) (For smooth, staggered, and scroll-triggered animations)
*   **Typography:** [Finlandica Headline](https://fonts.google.com/specimen/Finlandica) (Google Fonts)
*   **Icons:** [FontAwesome 6.5.1](https://fontawesome.com/) (Integrated via CDN)
*   **Theming:** `next-themes` (For seamless Dark/Light mode toggling)

## ✨ Key Features

*   **Glassmorphism UI:** Premium frosted glass effects, subtle borders, and dynamic glowing gradients.
*   **Fully Responsive:** A completely optimized mobile experience, including a professional full-screen blurred navigation overlay and touch-friendly targets.
*   **Dark/Light Mode:** Seamless theme switching with persistent state across reloads.
*   **GitHub Integration:** Live fetch of GitHub statistics (Public Repos, Followers, Stars) directly from the GitHub API.
*   **Micro-interactions:** Beautiful hover effects, scale transforms, and scroll-based entry animations utilizing Framer Motion.

## 💻 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

*   Node.js (v18 or higher recommended)
*   npm or yarn or pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/abhisheksinghcodebase/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the application.

## 📁 Project Structure

*   `/app`: Next.js App Router files (`layout.js`, `page.js`, `globals.css`)
*   `/components`: Modular React components (`Navbar.jsx`, `Hero.jsx`, `About.jsx`, `Projects.jsx`, `Skills.jsx`, `Contact.jsx`, `Footer.jsx`, `GitHubStats.jsx`)
*   `/public`: Static assets (Images, fonts, etc.)
